手機更新
兩種 OTA or 深層更新
Q:OTA(Over-the-Air)
A:基本上可以理解為熱更新(無縫更新) 安卓有分A/B兩個開機驅動 如果你再A 那OTA更新會更新B
Q:深層更新
A:就是升級安卓系統本身(14->15) 這種更新比較麻煩

OTA可以在更新完之後 不用重啟 先在另一邊把initboot.img檔案進行root作業後再重啟 及有root權限

深層更新就 重刷吧(重新刷寫boot)

可以對GPU進行降壓超頻(高通)(konabass)

A:以更低的電壓跑原本or更高的頻率 電壓越低則省電 但是過低開不了機
如果軟體or硬體設定錯誤
